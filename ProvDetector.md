# Running ProvDetector

## 1. Prerequisites

Install the required packages:

```bash
pip3 install pandas numpy networkx
```

## 2. Prepare Data

ProvDetector expects CSV files with 11 columns: `sourceId, sourceType, destinationId, destinationType, syscal, processName, retTime, pid, arg1, arg2, graphId`.

The repo's raw data files (in `StreamSpot-Evasion-Graphs/tajka/trainGraphs/`) only have 10 columns (no `graphId`), so you need to append a graph ID. Here's a helper script:

```python
# prepare_data.py
import os, csv

base = "StreamSpot-Evasion-Graphs"
trainDir = f"{base}/tajka/trainGraphs"
attackDir = f"{base}/attackGraphs"
evasionDir = f"{base}/evasion"

def combine_files(directory, files, start_gid, output_path):
    with open(output_path, 'w', newline='') as out:
        writer = csv.writer(out)
        for i, f in enumerate(sorted(files)):
            with open(os.path.join(directory, f)) as inp:
                for row in csv.reader(inp):
                    while len(row) < 10:
                        row.append('')
                    row.append(str(start_gid + i))
                    writer.writerow(row)

combine_files(trainDir, os.listdir(trainDir), 0, "/tmp/pd_train.csv")
combine_files(attackDir, os.listdir(attackDir)[:1], 100, "/tmp/pd_test_attack.csv")
combine_files(evasionDir, os.listdir(evasionDir)[:1], 200, "/tmp/pd_test_evasion.csv")
```

## 3. Run

Navigate to the provDetector directory and run the following commands:

```bash
cd provDetector

# Test attack graph against benign training data
python3 main.py /tmp/pd_train.csv /tmp/pd_test_attack.csv

# Test evasion graph against benign training data
python3 main.py /tmp/pd_train.csv /tmp/pd_test_evasion.csv
```

## 4. Output

It generates:
- `freqList.data` — frequency database (pickle)
- `setOfsets.data` — set of source/dest nodes per graph (pickle)
- `<trainfile>_kpathsTrainingGraphs.data` — top-20 anomalous paths (pickle)

## 5. Inspect Results

Use Python to inspect the results:

```python
import pickle
data = pickle.load(open('/tmp/pd_train.csv_kpathsTrainingGraphs.data', 'rb'))
for gname, kpaths in data:
    print(f"Graph {gname}: {len(kpaths)} anomalous paths")
    for kp in kpaths[:3]:
        print(f"  Score: {kp[1]:.2f}, Path length: {len(kp[0])}")
```

## Note

Delete `freqList.data` and `setOfsets.data` between runs with different training data, otherwise it reuses the cached frequency database:

```bash
rm -f freqList.data setOfsets.data
```

## Using the DARPA Theia Evasion Graph

A pre-built ProvDetector evasion graph for the DARPA TC3 Theia dataset is provided in `Theia-Evasion-Graphs/evasion-provDetector.csv` (also in `provDetectorTheia.zip`). It is already in the correct 11-column ProvDetector format.

This evasion graph was generated by injecting ~10K benign edges from the Theia dataset into the Drakon Firefox backdoor attack path. From the paper (§VI-F): a benign path with lower regularity than the attack paths was identified in the training data and injected 14–20 times, pushing the attack paths out of ProvDetector's top-K window.

To test it, you need the DARPA TC3 Theia benign data as training. Convert it using `parser/tcToProvParser.py`, then run:

```bash
cd provDetector
rm -f freqList.data setOfsets.data
python3 main.py <theia_benign_train.csv> ../Theia-Evasion-Graphs/evasion-provDetector.csv
```

Expected result: the evasion graph's attack paths are pushed out of the top-20 (K=20) anomalous paths, achieving 100% evasion.
